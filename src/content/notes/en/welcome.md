This repository is for recording what I study.

## How to add notes

Add markdown files under `src/content/notes/en/` and `src/content/notes/ko/`.

```md
---
title: 'Note title'
description: 'One-line summary'
pubDate: 2026-07-04
tags: ['tag1']
exploreNext:
  - label: 'Next topic'
    reason: 'Why it matters'
---

Write content here.
```

## Study Grass

Each note's `pubDate` appears on the Study Grass calendar on the home page.

## Languages

Every note has an English (`en/`) and Korean (`ko/`) version with the same filename (translation ID).
